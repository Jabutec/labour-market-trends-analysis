# Labour Market Trends & Business Impact Analysis (South Africa)

This project analyses South African unemployment data to provide insights for business decisions and workforce planning.

## Project Structure

- `data/raw/` – original datasets  
- `data/cleaned/` – cleaned / processed data  
- `sql/` – SQL queries for trend analysis  
- `notebooks/` – Python analysis using pandas  
- `visuals/charts/` – graphs and visualizations  
- `insights/` – business-focused implications from trends

## Goal

Translate labour market trends into actionable insights that help companies with hiring, workforce planning, and strategic decisions.
